# Streamlit MCP & LangGraph Chatbot - Math Task

This project demonstrates a **Streamlit-based chatbot** integrated with **MCP (Multi-Channel Processing)** and **LangGraph**, capable of handling **general math tasks** dynamically.

---

## Features

- **Streamlit Interface**: User-friendly web interface to interact with the chatbot.
- **MCP Integration**: Connects to MCP backend for executing task-specific operations.
- **LangGraph**: Used to manage task flows and process user inputs intelligently.
- **Math Tasks**: Supports arithmetic operations such as addition, subtraction, multiplication, division, and more.
- **Message History**: Tracks conversation history for smooth user experience.
